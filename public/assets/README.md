# Assets

Coloque aqui os arquivos visuais reais do jogo.

Estrutura sugerida:

- `characters/player/` para spritesheets do protagonista
- `characters/npcs/` para sprites ou retratos dos NPCs
- `maps/` para tilemaps JSON, tilesets e backgrounds
- `ui/` para icones, paineis e elementos de interface

Enquanto esses arquivos nao existem, o jogo usa texturas geradas em runtime.
